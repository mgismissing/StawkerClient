Stawker is a website created to get information from people in a totally legal way: by scraping it off of public posts on the internet.

It works with the supported websites' API and makes requests that get all of the information you need.

IMPORTANT:
This is the client version of Stawker, hence the name "Stawker Client".
Please use the server version when possible to access the full tool suite.
